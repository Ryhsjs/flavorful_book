# Вкусный букварь или Flavorful book

**Flavorful Book** - это веб-приложение для создания, 
поиска и обмена кулинарными рецептами с элементами социальной сети.

## О проекте

Flavorful Book позволяет пользователям:
- Создавать и публиковать собственные рецепты
- Искать рецепты по ингредиентам, категориям и времени приготовления
- Оставлять отзывы и оценки рецептам
- Добавлять понравившиеся рецепты в избранное
- Просматривать статистику популярности рецептов


## База данных

### Основные сущности
- **users** - пользователи системы
- **recipes** - рецепты
- **ingredients** - ингредиенты
- **categories** - категории рецептов
- **reviews** - отзывы и оценки
- **favorites** - избранные рецепты
- **recipe_views** - статистика просмотров

### Связи
- Многие-ко-многим: рецепты ↔ ингредиенты, рецепты ↔ категории
- Один-ко-многим: пользователь → рецепты, рецепт → отзывы

## Функциональность

### Аутентификация и авторизация
- Регистрация новых пользователей
- Вход/выход из системы
- Смена данных профиля

### Работа с рецептами
- **CRUD** операции над рецептами
- Поиск по названию, ингредиентам, категориям
- Фильтрация по времени приготовления
- Пагинация результатов
- Сортировка по дате создания, просмотрам, рейтингу и времени приготовления
